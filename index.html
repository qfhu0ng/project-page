Abstract (Short)

Mobile LLM agents inevitably interact with untrusted third-party content on smartphones. We study indirect prompt injection attacks where attacker-controlled content covertly steers agents away from benign user goals toward adversarial outcomes. We build an automated red-teaming pipeline combining scenario templates, injection patterns, and delivery channels, and ground failures using the MITRE ATT&CK Mobile framework.

Threat Model

  • Goal: induce the agent to disclose sensitive info, perform unintended cross-app actions, or install untrusted software.
  • Capability: deliver instructions via legitimate third-party channels (ads, notifications, webviews, emails/messages).
  • Constraint: no OS compromise, no weight tampering; actions must stay within normal mobile usage.

The core risk is a ‚Äúlethal trifecta‚Äù: access to rich sensitive data, ability to communicate externally, and exposure to untrusted inputs.

Automated Red-Teaming Pipeline

We generate diverse adversarial scenarios by composing: (1) scenario templates, (2) prompt-injection patterns, and (3) delivery channels. Runs are paired with benign executions for apples-to-apples comparison.

  • Channels: in-app ads, embedded web pages, email/messages, notifications
  • Patterns: context ignoring, task redirection, ‚Äúclick-to-continue‚Äù hooks, multi-hop chains
  • Labeling: outcomes grounded in MITRE ATT&CK Mobile tactics
